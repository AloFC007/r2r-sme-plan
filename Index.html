⚙️ SharePoint Configuration

This app connects directly to SharePoint Lists using your existing Microsoft 365 login. No Azure registration or IT admin needed.

You just need to create two lists on your SharePoint site: R2R_Sessions and R2R_PillarItems.

Your SharePoint Site
One-Time List Setup
  1. Go to your SharePoint site and click + New → List → Blank list
  2. Name it R2R_Sessions and add columns per the guide above
  3. Create another list named R2R_PillarItems and add its columns
  4. Paste your SharePoint site URL above and click Save & Connect
  5. Click ⬇ Pull from SharePoint anytime to get the latest team data
  6. Click ☁️ Sync to SharePoint after making changes to share with team
R2R_Sessions Columns Needed
Pillar (Choice) · SessionName (Text) · SessionDate (Date) · Owner (Text) · Participants (Text) · Attendees (Multi-line) · Agenda (Multi-line) · SessionStatus (Choice) · Notes (Multi-line)
R2R_PillarItems Columns Needed
Pillar (Choice) · ItemType (Choice) · ItemTitle (Text) · Description (Multi-line) · Owner (Text) · DueDate (Date) · Priority (Choice) · ItemStatus (Choice) · Tags (Text)
